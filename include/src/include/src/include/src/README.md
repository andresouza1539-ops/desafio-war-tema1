# Desafio War Estruturado – Tema 1

Projeto em C que implementa a lógica básica do jogo War com:
- Cadastro de territórios (vetor de structs)
- Ataque entre territórios utilizando ponteiros e alocação dinâmica
- Missões estratégicas (Ásia / América do Sul)
- Modularização (headers e fontes separados)

## Como usar (pelo site do GitHub)
1. Crie um novo repositório.
2. Add file → Create new file e cole cada arquivo conforme a estrutura indicada.
3. (Opcional) Baixe o projeto e compile localmente com `gcc` ou `make`.

## Compilação local (opcional)
```bash
gcc src/*.c -Iinclude -o war
./war
